人机交互机制

  • 1. 反馈性 (Feedback)： 任何表单提交都有顶部 Toast 状态浮窗；AI 响应包含骨架/打字等待态；作业截止日期自动计算并呈现橙红两级色彩警示。
  • 2. 一致性 (Consistency)： 全局共用统一色彩网格（蓝、靛、灰）、圆角尺度（12px/16px）、按钮响应形态以及完全一致的顶层平滑导航栏布局。
  • 3. 容错性 (Tolerance)： 新建表单含前端拦截与友好高亮提示；在输入无效问答时，系统提供快捷推荐引导方案而非白屏报错。
  • 4. 易学习性 (Learnability)： 首次载入包含明晰的欢迎引导气泡和四组典型用例卡片，允许用户一键体验高频场景。
  • 5. 可控性 (Controllability)： 桌面推送允许一键关闭；所有的历史归档、计划作业均带有“二次确认模态窗”支持随时可撤销性擦除。

内置支持的自然语言测试指令

“今天课表” / “上课”
调取输出当日课程安排数据。
“作业” / "截止"
盘点并输出所有未完成作业进度。
“考试” / “高数”
提取高数或特定学科考试安排。
“成绩” / “绩点”
调用当前激活学期成绩报表。
