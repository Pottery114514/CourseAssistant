
# 📝 ClassMate AI — 功能介绍与使用文档

ClassMate AI 是一款面向高校学生的**轻量级本地化学习管理工具**。项目采用纯前端技术栈，以自然语言交互（NLP）为核心体验，深度集成了课程、作业、自主计划和考试倒计时管理，旨在解决大学生日常信息分散、多学科截止日期记忆困难的痛点。

---

## 💡 第一部分：核心人机交互原则体现

本项目严格遵循人机交互（HCI）的五大核心设计原则，在无后端参与的纯前端原型中实现了媲美真实应用的高级交互体验：

* **反馈性 (Feedback)：**
* **打字等待：** AI 问答时提供仿真的骨架屏加载动画（`.dot-flashing`）。
* **全局通知：** 任何表单保存、状态勾选或单条删除，页面顶部都会在 1 秒内弹出动态 `Toast` 浮窗提示。
* **悬浮穿透：** 成绩柱状图支持鼠标悬浮（Hover）穿透显示精确分值与换算绩点。


* **一致性 (Consistency)：**
* **视觉规范：** 统一使用 `Tailwind CSS` 的 Slate/Blue 现代色彩网格、`12px/16px` 圆角规范。
* **导航统一：** 全局置顶的一致性导航，无论用户在哪个模块，都能以不超过 3 步的点击直达目标。


* **容错性 (Tolerance)：**
* **表单校验：** 空数据录入时触发前端拦截，并气泡提示错误原因。
* **多词义消除：** 当查询条件模糊（如输入“考试”却有多个科目）时，自动触发**歧义容错弹窗**，引导用户点击澄清，而不是直接报错或白屏。


* **易学习性 (Learnability)：**
* **零门槛新手引导：** AI 对话首页内置“今日课表”、“下周作业”等 4 个高频快捷用例卡片，新用户一键即可体验完整链路。


* **可控性 (Controllability)：**
* **防误删机制：** 清空历史、删除计划或考试时，必须通过“二次确认模态窗”的阻断校验，防止误触。
* **隐私与系统控制：** 桌面通知支持随时一键关闭，所有数据仅存储于用户的 `localStorage`，不上传网络，用户可随时一键清空，确保数据绝对可控。

## 🛠️ 第二部分：功能模块操作指南

### 1. 智能 AI 助手（自然语言问答交互）

* **对话与历史存档：** 在输入框内输入口语化的问题并回车，AI 引擎将自动进行模式匹配并答复，同时将对话自动归档至“历史问答”模块。
* **智能上下文级联（追问模式）：**
* **科目缓存：** 引擎自动缓存最近 3 轮对话里提到的科目名称（如：高等数学）。
* **省略式检测：** 当你使用“那”、“它”、“具体”、“然后”等词汇发起第二轮提问时（例如先问：“高等数学怎么考？”，再问：“**那**在哪里考？”），AI 会自动提取上一轮的科目补全问题，实现连贯的对话体验。


* **话题一键重置：** 输入框上方配有上下文状态栏。点击右侧红色的 **重置对话** 按钮并确认，可以一键清空上下文缓存，干净地开启新一轮话题。
* **主动话题切换：** 聊天时若输入“换个问题”、“聊点别的”、“换话题”等，系统会智能识别并自动在后台清空旧缓存，无需手动重置。
* **上下文来源标签：** 如果当前回答是基于上下文补全生成的，气泡上方会自动打上蓝色的 `[继承上下文话题: XX]` 标签，让机器人的思考路径清晰可见。

### 2. 课程查询模块

* **周视图模式：** 以宏观视角展示周一至周五的固定课程表，课表卡片内包含：科目、任课教师、教室地点（加粗高亮）。
* **今日看板模式：** 一键切换为“仅看今天”，系统自动抓取当前是星期几，并按时间线单列瀑布流展示今天的课程，方便课前在手机或设备上快速查阅。

### 3. 计划与作业管理

* **作业截止（双级智能高亮预警）：**
* **橙色预警：** 距离截止日期（Deadline）不足 3 天的作业，卡片自动变为橙色背景。
* **红色高亮：** 当天截止或超期的作业，卡片变为红色背景，且带有关联标签闪烁提醒，产生视觉紧迫感。
* **联动通知：** 勾选完成作业时，如果开启了右侧的“桌面弹窗提醒”，浏览器将联动发送系统的原生横幅通知。


* **成长计划：** 可录入课外的自主复习计划（包含预计时长），支持一键打勾归档。

### 4. 成绩大屏与可视化

* **动态数据联动：** 顶部支持下拉菜单切换“学期”（如 2025-秋季学期 / 2026-春季学期）。
* **核心 KPI 看板：** 自动实时计算当前学期的“平均分”、根据标准公式换算的“预计综合 GPA”以及学习计划的“总完成率”。
* **手绘图表：** 左侧为各科目的考试成绩柱状对比图（带悬浮数值提示），右侧为成长计划达成率的环形比例图，数据切换时均有平滑的渲染过渡。

### 5. 期末考试倒计时（内嵌）

* **考试日程录入：** 在课表页面下方，点击“录入新考试”，可填写科目、日期、考场地点及学期。
* **阶梯级时限卡片：**
* 距离考试 **>7天**：显示普通白色卡片。
* 距离考试 **3~7天**：显示黄色背景（临近预警）。
* 距离考试 **<3天**：显示红色背景（极度紧迫）。
* 考试日期 **已过**：显示“⚠️ 已过期”置灰卡片。


* **数据动态置顶：** 卡片根据考试日期的先后顺序自动由近到远排序，列表顶部实时统计并更新“最近一次考试的科目与倒计时天数”。

### 6. 历史对话档案

* **全局检索：** 支持在输入框内实时增量检索历史对话的关键词。
* **高效复用：** 每一条历史记录右上角均提供 **“复用问题”** 按钮，点击后系统会自动帮你把该问题填回 AI 助手的输入框并跳转，极大减少重复输入。

## 🚀 第三部分：典型测试用例示例（用于功能演示）

为了在原型评审或答辩中展现最佳交互效果，建议按照以下**演练剧本**进行操作展示：

1. **首屏易学性：** 进入页面，在 AI 助手首页直接点击快捷卡片 **“今天有什么课？”** $\rightarrow$ 观察打字加载动画 $\rightarrow$ 观察右侧摘要区同步刷新。
2. **上下文与追问：**
* 第一步输入：`高等数学`。AI 将为你展示高数的信息。此时观察**输入框上方的状态栏**，已由“无”变为“当前话题：高等数学(下)”。
* 第二步输入省略式问题：`具体在哪里考？`。AI 将智能识别出开头带有“具体”，自动继承话题，为你输出高等数学在“第十五教学楼 301”的考试地点，同时气泡上方出现**蓝色继承标签**。


3. **歧义容错检测：** 在输入框中故意输入一个模糊的词汇：`考试时间`。因为系统中同时预置了“高等数学”与“数据结构”的考试，页面中央会立刻**淡入弹出科目选择弹窗**。点击“数据结构基础”，输入框自动补全并重新精准查询。
4. **主动话题清理：** 接着输入：`换个话题，我还有作业吗？`。系统检测到“换个话题”，聊天区内立即插入一条灰色系统消息：`🔄 检测到话题切换，已自动清空上下文`，同时无缝回答关于作业的提问，状态栏恢复清空。
5. **主页考试弹窗提醒：**
* 切换到“课程表”选项卡，拉到最下方，点击“录入新考试”。
* 将考试日期选择为**今天**（当前日期），科目填写“计算机网络原理”，点击保存。
* **刷新网页** $\rightarrow$ 页面首屏将立刻弹出一个带有跳动铃铛图标的**黄金色高亮大弹窗**，提示你`今日考试提醒！今天有需要参加的期末考试...`，充分体现到期双重预警的 HCI 设计原则。
